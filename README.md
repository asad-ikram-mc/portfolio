# Asad Ikram — Portfolio

Personal portfolio website for Asad Ikram, Full Stack Data Engineer & Chevening Scholar.

## Deploy to GitHub Pages

1. Create a new GitHub repo named `asad-ikram.github.io` (or any name)
2. Push this folder's contents to the `main` branch
3. Go to **Settings → Pages → Source → Deploy from branch → main → / (root)**
4. Your site will be live at `https://yourusername.github.io`

## Or deploy with custom domain

Point your domain's DNS to GitHub Pages and add a `CNAME` file:
```
echo "yourdomain.com" > CNAME
```

## Local preview

Just open `index.html` in any browser — no build step needed.
